# Quizboard

![Version](https://img.shields.io/github/package-json/v/Braincoke24/QuizBoard)
![License](https://img.shields.io/github/license/Braincoke24/QuizBoard)
![Status](https://img.shields.io/badge/status-work_in_progress-orange)

A lightweight, browser-based quiz game inspired by classic TV quiz formats.
Designed for local multiplayer with minimal setup.

## Quick Start

```bash
npm install
npm run dev
```

---

## Features

- Customizable quiz board with categories and point values
- Multiple players with live score tracking
- Mouse-based and optional keyboard-based buzzer input
- Configurable game rules and score multipliers
- Fully client-side, no backend required

---

## Tech Stack

- **TypeScript**
- **Vite** for development and build tooling
- **Vanilla DOM APIs** (no frontend framework)
- **Radix UI Colors** for theming
- **Vitest** for testing

---

## Getting Started

### Prerequisites

- Node.js (version 18 or newer recommended)
- npm

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```

The production build outputs static files and can be hosted on any static web server.

---

## Deployment

Quizboard is a static web application built with Vite.
The generated dist/ directory can be deployed to any static hosting provider.

Example setups include:

- Traditional web hosting
- Static file servers
- CI-based deployments (e.g. GitHub Actions)
